# SORTME
SortMe is a React App project by Aadeesh Bali to visually compare the sorting process, speed, and effeciency of the six most common data sorting algorithms

# ALGORITHMS
-Bubble Sort <br/>
-Selection Sort <br/>
-Insertion Sort <br/>
-Merge Sort <br/> 
-Quick Sort <br/>
-Heap Sort <br/>
